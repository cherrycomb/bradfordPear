Charlotte Ornamental Pear Density

This map shows an example of using the leaflet.heat API for creating a density heatmap of point data (in this case, the 9,823 ornamental pear trees recorded in Charlotte's Tree Canopy Inventory).

About the data

The data visualized in this map comes from Charlotte's official urban canopy inventory.
